## Product in one sentence

SarkariExamsAI is a **Knowledge Intelligence Platform** for government exam preparation — not an AI chatbot.

**Philosophy:** *One Goal → One Screen → One Decision.*

---

## What students get

| Experience | Description |
|------------|-------------|
| **Topic Learning Workspace** | One screen: read + exam intelligence + highlights + next step |
| **Practice** | PYQ-pattern-aware MCQs linked to what they read |
| **Revision** | Priority driven by PYQ frequency + weakness |
| **Navigator** (future) | Graph-based "what to read next" |

---

## Target exams (phased)

| Phase | Exams |
|-------|-------|
| 1 | BPSC Prelims + Mains GS-1 |
| 2 | UPSC, State PSC |
| 3 | SSC, Banking, Railways |

**Platform IP:** Same intelligence engine — new exams = new content + question bank ingestion.

## Success metrics

| Metric | 12-month target |
|--------|-----------------|
| Topic completion rate | > 60% |
| D7 retention | > 35% |
| Practice accuracy lift after reading | +15% |
| PYQ tag coverage (flagship topics) | > 80% |
| Content coverage | 3 subjects × NCERT classes |
